# 쇼미더머니 — 개발 정리 문서 (최종)

> 주식투자를 테마로 한 캐주얼 아케이드 웹 게임. 캐릭터가 좌우로 움직이며 **양봉은 먹고, 음봉은 피해** 제한시간 동안 투자금을 최대한 불린다.
> 파일: `index.html` (단일 HTML, 별도 서버 불필요 / 리더보드만 Supabase 사용)

---

## 1. 개요
- **제목**: 불장을 기다리며 (시작 화면 로고, "**불장**" 초록 강조)
- **플랫폼**: 모바일·PC 웹 브라우저 (단일 `index.html`)
- **한 판**: 90초 / **시작 투자금**: 3,000만 원
- **목표**: 종료 시점의 투자금이 최종 스코어 → 등급(칭호) 부여 + 리더보드 등록
- **게임오버**: 제한시간 종료, 또는 투자금 0원(파산)

## 2. 핵심 규칙
- 위에서 캔들이 떨어지고, 플레이어는 **좌우로만** 이동.
- 🟩 **양봉(초록)** 먹으면 투자금 증가, 🟥 **음봉(빨강)** 맞으면 감소. (미국식 색상)
- 캔들의 **몸통뿐 아니라 위·아래 꼬리(심지)에 닿아도** 먹힌다.
- 시간이 지날수록 낙하 속도·스폰 밀도 증가(초반부터 빠른 템포).
- 투자금은 **다이나믹 카운트업**으로 굴러가고, 획득/손실 시 숫자가 커졌다 색이 번쩍(초록/빨강).
- **조작**: 터치 드래그(손가락 1:1 추적, 세밀 조정) / 방향키(계속 누르면 **가속** — 최고 W×2.4). 시작 화면 안내 "조작방법 : 방향키 or 마우스 드래그".

## 3. 캔들 시스템
- 실제 캔들스틱 패턴 기반 **12단계**: 십자형·잠자리형·비석형·망치형·역망치형·팽이형·유성형·교수형·중형·장대·**초장대·나락**.
- 위·아래 꼬리·몸통 길이가 매번 랜덤 → 같은 단계도 모양 다양. 캔들은 단색.
- 단계↑ = 길고 증감액 큼. **긴 양봉일수록 폭이 좁아** 조준 난도↑.
- **하락 큰 음봉(11·12단계)**: 중반(50%) 이후 등장. 초장대음봉 −1,800만 / 나락음봉 −2,700만.

## 4. 특수 캔들 · 이벤트
### 상한가 / 하한가 (희귀)
- **상한가(↑30%, 초록·깜빡임·"30%")**: 현재 투자금 +30%.
- **하한가(↓30%, 빨강·깜빡임)**: −30%. 폭이 5단계 랜덤(넓을수록 회피 난도↑). 상한가보다 약간 더 자주 등장.

### 🚀 불장 타임 (20콤보)
- 양봉 20연속으로 발동(음봉에 맞아 콤보 끊긴 뒤 재무장 → 무한 연장 방지).
- 5초간: **상한가는 유지**, **하한가·일반 캔들은 초록 양봉으로 전환**되고 이후 양봉만 쏟아짐. 발동 즉시 **상한가 2개 확정 스폰**.
- 연출: 초록 플래시 → "🚀 가즈아!" 배너 → 초록 테두리 글로우 + "🔥 불장 타임 X.Xs" 카운트다운.
- **BGM 전환**: 불장 동안 기본 BGM 대신 **밝고 경쾌한 오리지널 "무적" 멜로디**(C장조·빠른 템포)로 교체, 종료 시 원복.

### 📉 폭락장 (한 판 2회)
- 중·후반 2번(대략 38~50초, 62~76초) 발동. 5.5초간 음봉·하한가·나락음봉만 쏟아짐.
- 붉은 화면 맥동 + 테두리 번쩍 + 강한 흔들림 + "📉 폭락장 X.Xs" 카운트다운.
- **탈출구**: 폭락장 중 20콤보로 불장을 터뜨리면 즉시 해제(불장 우선).
- **양끝 캠핑 방지**: 폭락장·떡락 뉴스 중 음봉의 약 40%가 플레이어 위치 쪽으로 살짝 조준(좌우 퍼짐 있어 이동하면 회피 가능, 나머지는 전 구간 랜덤).

### 📰 NEWS 아이템 (한 판 3~5회)
- 보라색으로 흔들리는 신문(물음표 미스터리) — **먹기 전엔 호재/악재 모름**.
- 먹으면 확정, 배너 색으로 표시: **🚀 떡상 뉴스(초록)** / **💣 떡락 뉴스(빨강)**.
  - 떡상: 5초간 양봉 확률 90% + **초광폭 상한가 1~2개** 등장.
  - 떡락: 5초간 음봉 확률 90% + **초광폭 하한가 1~2개** 등장.
- 불장 중에는 뉴스 대기(이벤트 겹침 방지).
- 먹는 순간 배너 아래 **패러디 가짜 헤드라인 1개 랜덤 노출**(약 2.8초, 「 」 안, 떡상 초록·떡락 빨강). 배너는 살짝 작게(40px), 헤드라인은 21px.
  - 각 문구에 이모지 포함. 떡상 8개(📈 어닝 서프라이즈 · ⭐ 일론 머쓱 · 🚗 헐대차 BMV · 📦 쿠빵 · 💊 삼송바이오 · 🍽️ 잭슨황 식사 · 💰 국민연금 올인 · 🏛️ 매도버튼 삭제법), 떡락 7개(💥 외국인 매도 · 😱 까까오 상폐 · 💻 항국은행 해킹 · 🥊 트럼푸 X파일 · ⚛️ 초전도 양자컴터 · 🚨 계엄령 · 🍗 잭슨황 치킨).
  - 문구는 코드 상단 `BOOM_NEWS` / `FAKE_NEWS` 배열에서 관리. **한 판 내 중복 없음**(시작 시 섞어 가방에 담아 하나씩 소진, 떡상 7 · 떡락 7).

## 6. 캐릭터 (등급별 8단계 변신)
- 벡터(코드) 캐릭터. **투자금 등급이 오를수록 좀비 → 부자 → 재벌로 실시간 변신**, 파산 쪽으로 내려가면 다시 좀비로.
  1. 🧟 **좀비** (~5,000만, 시작 지점): 창백한 녹빛 피부·다크서클, 해진 티·찢어진 청바지
  2. 후줄근 (5,000만+): 칙칙한 회색 티셔츠
  3. 평범 (1억+): 파란 티셔츠 + 청바지 + 운동화
  4. 직장인 (2억+): 드레스 셔츠 + 넥타이 + 안경
  5. 정장 (4억+): 네이비 슈트 + 넥타이 + 구두
  6. 선글라스 부자 (8억+): 슈트 + 선글라스
  7. 재벌 (20억+): 블랙&골드 슈트 + 금 넥타이 + 선글라스
  8. 👑 갓재벌 (80억+): 백금 정장 + 금 넥타이·구두 + 선글라스 + 왕관 + 반짝이
- **이벤트별 표정 반응**: 양봉=미소 / 음봉=찡그림+식은땀 / 상한가=함박웃음+반짝이 / 하한가=놀란 눈+식은땀 / 불장=$ 눈.
- 미세한 idle 흔들림, 바닥 그림자. 모바일 하단 UI에 안 가리도록 하단에서 ~11% 위 배치.

## 7. 리더보드
- **닉네임 입력**(최대 10자, 자동 기억) 후 시작.
- **인트로·결과 모두 TOP 10** 노출(타이틀 "🏆 TOP PLAYERS"). **탭 UI로 2종 전환**(넓은 알약형, 활성 파란색): 💰 점수(돈) / 🔥 콤보(최고 콤보).
  - **항상 10칸 채움**: 실제 기록이 부족하면 재치있는 가짜 유저로 채움(점수 140억~1500만 로그 분포로 등급 전반에 고르게, `FAKE_NAMES`/`FAKE_LB`). 실제 점수는 정상 정렬돼 섞임.
  - 리스트 자체 스크롤 없이 **화면 전체가 스크롤**(스크롤바 숨김).
  - 점수 탭: `순위 · 닉네임 (칭호) · 금액` / 콤보 탭: `순위 · 닉네임 (칭호) · N 콤보`.
  - 내 순위는 초록 강조(굵기는 빼서 최종 금액보다 커 보이지 않게). (결과 화면은 결과 정보와 리더보드 사이 여백 확대)
- **Supabase 공유 랭킹** 사용(점수=`order=score.desc`, 콤보=`order=combo.desc`). 설정값 없으면 로컬(localStorage)로 자동 대체.
- 테이블/정책(SQL):
```sql
create table scores ( id bigint generated always as identity primary key,
  name text, score bigint, combo int, created_at timestamptz default now() );
alter table scores enable row level security;
create policy "read"   on scores for select using (true);
create policy "insert" on scores for insert with check (true);
```
- 코드 상단 `SB_URL` / `SB_KEY`(anon public)에 값 입력.
- **랭킹 초기화**: SQL Editor `truncate table scores;` 또는 Table Editor 개별 삭제.

## 8. 사운드
- **Web Audio API로 즉석 생성** — 음원 파일 없이 단일 HTML 유지.
- 효과음: **양봉(콤보마다 C장조 음계를 한 음씩 밟고 상승 + 5도 하모니·옥타브 반짝임·저음 바디, 10콤보마다 팡파레)** / 음봉(하강) / 상한가(상승 아르페지오) / 하한가(하강) / 🚀불장(팡파레) / 📉폭락장(저음 사이렌) / 📰뉴스 등장(신비로운 알림) · 뉴스 획득음(떡상=상승/떡락=하강으로 구분) / 게임오버(마무리 멜로디) / 버튼 클릭.
- **BGM**: 게임 분위기에 맞춘 **오리지널 A단조 그루브**(리듬 리프 + 하이햇 + 킥). **불장 타임에는 경쾌한 C장조 무적 멜로디로 전환**. Web Audio로 즉석 생성, 게임 시작 시 재생·종료 시 정지. 마스터 저음필터로 부드럽게.
- **음소거 버튼**: 우측 상단 🔊 → 🔇 토글(효과음+BGM). 오디오는 모바일 정책상 첫 터치(게임 시작)에서 활성화.
- **볼륨 밸런싱**: 마스터 0.95, BGM 각 파트 볼륨 상향(멜로디·베이스·킥). 양봉음은 두툼하게(메인+사각파+저음 레이어).

## 9. 배포 (GitHub Pages)
1. Public 저장소 생성 → 루트에 `index.html` **및 `og.jpg`(링크 썸네일)** 업로드(Add file → Upload files → Commit).
2. Settings → Pages → Source: Deploy from a branch, Branch: main / (root) → Save.
3. `https://<아이디>.github.io/<저장소>/` 로 공개. 반영은 보통 1~2분(Actions 탭 초록 체크로 확인).
4. 수정 재배포: 같은 이름으로 다시 Upload → Commit. 안 바뀌면 강력 새로고침 또는 `?v=2` 붙이기.
5. **링크 미리보기(OG)**: `<head>`에 Open Graph 메타태그 + 절대경로 `og.jpg`(1200×630, 포스터 재구성) 지정. 카카오 등은 미리보기를 캐시하므로 갱신 시 `?v=2` 등으로 우회.

## 10. 백로그 (미구현 아이디어)
- 배당금 코인, 서킷브레이커
- 아이템/파워업: 레버리지, 손절 방패, 존버 모드(무적), 공매도(조작 반전)
- 결과 화면 부가 통계(먹은 상한가 수 등)
- 치팅 방지(점수 검증/난독화), 닉네임 중복 시 최고점만 유지

## 11. 밸런싱 값 요약 (조정 포인트)
- 시작 3,000만 / 90초
- 캔들 증가액(만원): 5·15·30·60·120·200·350·550·800·1200 (+음봉 1800·2700)
- 상한가·하한가 ±30% / 불장 20콤보·5초 / 폭락장 5.5초·2회 / 뉴스 3~5회
- 특수 확률, 낙하 속도·스폰 간격은 `spawnCandle`·`difficulty()`에서 조정
- 결과 점수는 길어져도 **한 줄 자동 맞춤**(폰트 40px→최소 18px 축소).
